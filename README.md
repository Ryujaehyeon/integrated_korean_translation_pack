# Stellaris 통합 한국어 번역 팩

Stellaris 모드 50여 개의 한국어 번역을 하나의 팩으로 통합한 모드입니다.

## 포함 모드

- Gigastructural Engineering & More
- Merged Leader Levels
- NSC3 Season 1
- Extra Ship Components NEXT
- Plentiful Traditions
- Expanded Stellaris Traditions / Ascension Perks
- Unique Ascension Perks
- Planetary Diversity
- UI Overhaul Dynamic
- More Events Mod
- 그 외 다수

## 설치

1. 이 레포의 최신 릴리즈에서 `integrated_korean_translation_pack.zip` 다운로드
2. 압축 해제 후 `integrated_korean_translation_pack` 폴더를 Stellaris mod 폴더에 복사
   - Windows: `%USERPROFILE%\Documents\Paradox Interactive\Stellaris\mod\`
3. Stellaris 런처에서 번역 팩 활성화

## 번역 도구

번역 작업에 사용된 도구: [translation-tools](https://github.com/Ryujaehyeon/translation-tools)
